<h1 class="firstHeading" id="firstHeading" lang="en">Composite pattern</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><p>In <a href="/wiki/Software_engineering" title="Software engineering">software engineering</a>, the <b>composite pattern</b> is a partitioning <a class="mw-redirect" href="/wiki/Design_pattern_(computer_science)" title="Design pattern (computer science)">design pattern</a>. The composite pattern describes that a group of objects is to be treated in the same way as a single instance of an object. The intent of a composite is to "compose" objects into tree structures to represent part-whole hierarchies. Implementing the composite pattern lets clients treat individual objects and compositions uniformly.<sup class="reference" id="cite_ref-GangOfFour_1-0"><a href="#cite_note-GangOfFour-1">[1]</a></sup></p>

<h2><span class="mw-headline" id="Motivation">Motivation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=1" title="Edit section: Motivation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>When dealing with Tree-structured data, programmers often have to discriminate between a leaf-node and a branch. This makes code more complex, and therefore, error prone. The solution is an interface that allows treating complex and primitive objects uniformly. In <a href="/wiki/Object-oriented_programming" title="Object-oriented programming">object-oriented programming</a>, a composite is an object designed as a composition of one-or-more similar objects, all exhibiting similar functionality. This is known as a "<a href="/wiki/Has-a" title="Has-a">has-a</a>" relationship between objects.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> The key concept is that you can manipulate a single instance of the object just as you would manipulate a group of them. The operations you can perform on all the composite objects often have a <a class="mw-redirect" href="/wiki/Least_common_denominator" title="Least common denominator">least common denominator</a> relationship. For example, if defining a system to portray grouped shapes on a screen, it would be useful to define resizing a group of shapes to have the same effect (in some sense) as resizing a single shape.</p>
<h2><span class="mw-headline" id="When_to_use">When to use</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=2" title="Edit section: When to use">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Composite should be used when clients ignore the difference between compositions of objects and individual objects.<sup class="reference" id="cite_ref-GangOfFour_1-1"><a href="#cite_note-GangOfFour-1">[1]</a></sup> If programmers find that they are using multiple objects in the same way, and often have nearly identical code to handle each of them, then composite is a good choice; it is less complex in this situation to treat primitives and composites as homogeneous.</p>
<h2><span class="mw-headline" id="Structure">Structure</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=3" title="Edit section: Structure">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<dl>
<dt>Component</dt>
</dl>
<ul>
<li>is the abstraction for all components, including composite ones</li>
<li>declares the interface for objects in the composition</li>
<li>(optional) defines an interface for accessing a component's parent in the recursive structure, and implements it if that's appropriate</li>
</ul>
<dl>
<dt>Leaf</dt>
</dl>
<ul>
<li>represents leaf objects in the composition</li>
<li>implements all Component methods</li>
</ul>
<dl>
<dt>Composite</dt>
</dl>
<ul>
<li>represents a composite Component (component having children)</li>
<li>implements methods to manipulate children</li>
<li>implements all Component methods, generally by delegating them to its children</li>
</ul>

<h2><span class="mw-headline" id="Variation">Variation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=4" title="Edit section: Variation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>As it is described in <a href="/wiki/Design_Patterns" title="Design Patterns">Design Patterns</a>, the pattern also involves including the child-manipulation methods in the main Component interface, not just the Composite subclass. More recent descriptions sometimes omit these methods.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup></p>
<h2><span class="mw-headline" id="Example">Example</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=5" title="Edit section: Example">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The following example, written in <a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a>, implements a graphic class, which can be either an ellipse or a composition of several graphics. Every graphic can be printed. In <a class="mw-redirect" href="/wiki/Backus-Naur_form" title="Backus-Naur form">Backus-Naur form</a>,</p>
<pre>
       Graphic ::= ellipse | GraphicList
       GraphicList ::= empty | Graphic GraphicList
</pre>
<p>It could be extended to implement several other shapes (rectangle, etc.) and methods (<a href="/wiki/Translation_(geometry)" title="Translation (geometry)">translate</a>, etc.).</p>
<h3><span class="mw-headline" id="Java">Java</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=6" title="Edit section: Java">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="cm">/** "Component" */</span>
<span class="kd">interface</span> <span class="nc">Graphic</span> <span class="o">{</span>

    <span class="c1">//Prints the graphic.</span>
    <span class="kd">public</span> <span class="kt">void</span> <span class="nf">print</span><span class="o">();</span>
<span class="o">}</span>

<span class="cm">/** "Composite" */</span>
<span class="kn">import</span> <span class="nn">java.util.List</span><span class="o">;</span>
<span class="kn">import</span> <span class="nn">java.util.ArrayList</span><span class="o">;</span>
<span class="kd">class</span> <span class="nc">CompositeGraphic</span> <span class="kd">implements</span> <span class="n">Graphic</span> <span class="o">{</span>

    <span class="c1">//Collection of child graphics.</span>
    <span class="kd">private</span> <span class="n">List</span><span class="o">&lt;</span><span class="n">Graphic</span><span class="o">&gt;</span> <span class="n">childGraphics</span> <span class="o">=</span> <span class="k">new</span> <span class="n">ArrayList</span><span class="o">&lt;</span><span class="n">Graphic</span><span class="o">&gt;();</span>

    <span class="c1">//Prints the graphic.</span>
    <span class="kd">public</span> <span class="kt">void</span> <span class="nf">print</span><span class="o">()</span> <span class="o">{</span>
        <span class="k">for</span> <span class="o">(</span><span class="n">Graphic</span> <span class="n">graphic</span> <span class="o">:</span> <span class="n">childGraphics</span><span class="o">)</span> <span class="o">{</span>
            <span class="n">graphic</span><span class="o">.</span><span class="na">print</span><span class="o">();</span>
        <span class="o">}</span>
    <span class="o">}</span>

    <span class="c1">//Adds the graphic to the composition.</span>
    <span class="kd">public</span> <span class="kt">void</span> <span class="nf">add</span><span class="o">(</span><span class="n">Graphic</span> <span class="n">graphic</span><span class="o">)</span> <span class="o">{</span>
        <span class="n">childGraphics</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">graphic</span><span class="o">);</span>
    <span class="o">}</span>

    <span class="c1">//Removes the graphic from the composition.</span>
    <span class="kd">public</span> <span class="kt">void</span> <span class="nf">remove</span><span class="o">(</span><span class="n">Graphic</span> <span class="n">graphic</span><span class="o">)</span> <span class="o">{</span>
        <span class="n">childGraphics</span><span class="o">.</span><span class="na">remove</span><span class="o">(</span><span class="n">graphic</span><span class="o">);</span>
    <span class="o">}</span>
<span class="o">}</span>

<span class="cm">/** "Leaf" */</span>
<span class="kd">class</span> <span class="nc">Ellipse</span> <span class="kd">implements</span> <span class="n">Graphic</span> <span class="o">{</span>

    <span class="c1">//Prints the graphic.</span>
    <span class="kd">public</span> <span class="kt">void</span> <span class="nf">print</span><span class="o">()</span> <span class="o">{</span>
        <span class="n">System</span><span class="o">.</span><span class="na">out</span><span class="o">.</span><span class="na">println</span><span class="o">(</span><span class="s">"Ellipse"</span><span class="o">);</span>
    <span class="o">}</span>
<span class="o">}</span>

<span class="cm">/** Client */</span>
<span class="kd">public</span> <span class="kd">class</span> <span class="nc">Program</span> <span class="o">{</span>

    <span class="kd">public</span> <span class="kd">static</span> <span class="kt">void</span> <span class="nf">main</span><span class="o">(</span><span class="n">String</span><span class="o">[]</span> <span class="n">args</span><span class="o">)</span> <span class="o">{</span>
        <span class="c1">//Initialize four ellipses</span>
        <span class="n">Ellipse</span> <span class="n">ellipse1</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Ellipse</span><span class="o">();</span>
        <span class="n">Ellipse</span> <span class="n">ellipse2</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Ellipse</span><span class="o">();</span>
        <span class="n">Ellipse</span> <span class="n">ellipse3</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Ellipse</span><span class="o">();</span>
        <span class="n">Ellipse</span> <span class="n">ellipse4</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Ellipse</span><span class="o">();</span>

        <span class="c1">//Initialize three composite graphics</span>
        <span class="n">CompositeGraphic</span> <span class="n">graphic</span> <span class="o">=</span> <span class="k">new</span> <span class="n">CompositeGraphic</span><span class="o">();</span>
        <span class="n">CompositeGraphic</span> <span class="n">graphic1</span> <span class="o">=</span> <span class="k">new</span> <span class="n">CompositeGraphic</span><span class="o">();</span>
        <span class="n">CompositeGraphic</span> <span class="n">graphic2</span> <span class="o">=</span> <span class="k">new</span> <span class="n">CompositeGraphic</span><span class="o">();</span>

        <span class="c1">//Composes the graphics</span>
        <span class="n">graphic1</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">ellipse1</span><span class="o">);</span>
        <span class="n">graphic1</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">ellipse2</span><span class="o">);</span>
        <span class="n">graphic1</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">ellipse3</span><span class="o">);</span>

        <span class="n">graphic2</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">ellipse4</span><span class="o">);</span>

        <span class="n">graphic</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">graphic1</span><span class="o">);</span>
        <span class="n">graphic</span><span class="o">.</span><span class="na">add</span><span class="o">(</span><span class="n">graphic2</span><span class="o">);</span>

        <span class="c1">//Prints the complete graphic (four times the string "Ellipse").</span>
        <span class="n">graphic</span><span class="o">.</span><span class="na">print</span><span class="o">();</span>
    <span class="o">}</span>
<span class="o">}</span>

<h3><span class="mw-headline" id="C.23">C#</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Composite_pattern&amp;action=edit&amp;section=7" title="Edit section: C#">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The following example, written in <a href="/wiki/C_Sharp_(programming_language)" title="C Sharp (programming language)">C#</a>.</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="k">namespace</span> <span class="nn">CompositePattern</span>
<span class="p">{</span>
    <span class="k">using</span> <span class="nn">System</span><span class="p">;</span>
    <span class="k">using</span> <span class="nn">System.Collections.Generic</span><span class="p">;</span>
    <span class="k">using</span> <span class="nn">System.Linq</span><span class="p">;</span>
    <span class="c1">//Client</span>
    <span class="k">class</span> <span class="nc">Program</span>
    <span class="p">{</span>
        <span class="k">static</span> <span class="k">void</span> <span class="nf">Main</span><span class="p">(</span><span class="kt">string</span><span class="p">[]</span> <span class="n">args</span><span class="p">)</span>
        <span class="p">{</span>
            <span class="c1">// initialize variables</span>
            <span class="kt">var</span> <span class="n">compositeGraphic</span> <span class="p">=</span> <span class="k">new</span> <span class="n">CompositeGraphic</span><span class="p">();</span>
            <span class="kt">var</span> <span class="n">compositeGraphic1</span> <span class="p">=</span> <span class="k">new</span> <span class="n">CompositeGraphic</span><span class="p">();</span>
            <span class="kt">var</span> <span class="n">compositeGraphic2</span> <span class="p">=</span> <span class="k">new</span> <span class="n">CompositeGraphic</span><span class="p">();</span>

            <span class="c1">//Add 1 Graphic to compositeGraphic1</span>
            <span class="n">compositeGraphic1</span><span class="p">.</span><span class="n">Add</span><span class="p">(</span><span class="k">new</span> <span class="n">Ellipse</span><span class="p">());</span>

            <span class="c1">//Add 2 Graphic to compositeGraphic2</span>
            <span class="n">compositeGraphic2</span><span class="p">.</span><span class="n">AddRange</span><span class="p">(</span><span class="k">new</span> <span class="n">Ellipse</span><span class="p">(),</span> 
                <span class="k">new</span> <span class="nf">Ellipse</span><span class="p">());</span>

            <span class="cm">/*Add 1 Graphic, compositeGraphic1, and </span>
<span class="cm">              compositeGraphic2 to compositeGraphic */</span>
            <span class="n">compositeGraphic</span><span class="p">.</span><span class="n">AddRange</span><span class="p">(</span><span class="k">new</span> <span class="n">Ellipse</span><span class="p">(),</span> 
                <span class="n">compositeGraphic1</span><span class="p">,</span> 
                <span class="n">compositeGraphic2</span><span class="p">);</span>

            <span class="cm">/*Prints the complete graphic </span>
<span class="cm">            (four times the string "Ellipse").*/</span>
            <span class="n">compositeGraphic</span><span class="p">.</span><span class="n">Print</span><span class="p">();</span>
            <span class="n">Console</span><span class="p">.</span><span class="n">ReadLine</span><span class="p">();</span>
        <span class="p">}</span>
    <span class="p">}</span>
    <span class="c1">//Component</span>
    <span class="k">public</span> <span class="k">interface</span> <span class="n">IGraphic</span>
    <span class="p">{</span>
        <span class="k">void</span> <span class="nf">Print</span><span class="p">();</span>
    <span class="p">}</span>
    <span class="c1">//Leaf</span>
    <span class="k">public</span> <span class="k">class</span> <span class="nc">Ellipse</span> <span class="p">:</span> <span class="n">IGraphic</span>
    <span class="p">{</span>
        <span class="c1">//Prints the graphic</span>
    	<span class="k">public</span> <span class="k">void</span> <span class="nf">Print</span><span class="p">()</span>
        <span class="p">{</span>
            <span class="n">Console</span><span class="p">.</span><span class="n">WriteLine</span><span class="p">(</span><span class="s">"Ellipse"</span><span class="p">);</span>
        <span class="p">}</span>
    <span class="p">}</span>
    <span class="c1">//Composite</span>
    <span class="k">public</span> <span class="k">class</span> <span class="nc">CompositeGraphic</span> <span class="p">:</span> <span class="n">IGraphic</span>
    <span class="p">{</span>
        <span class="c1">//Collection of Graphics.</span>
        <span class="k">private</span> <span class="k">readonly</span> <span class="n">List</span><span class="p">&lt;</span><span class="n">IGraphic</span><span class="p">&gt;</span> <span class="n">graphics</span><span class="p">;</span>

        <span class="c1">//Constructor </span>
        <span class="k">public</span> <span class="nf">CompositeGraphic</span><span class="p">()</span>
        <span class="p">{</span>
            <span class="c1">//initialize generic Collection(Composition)</span>
            <span class="n">graphics</span> <span class="p">=</span> <span class="k">new</span> <span class="n">List</span><span class="p">&lt;</span><span class="n">IGraphic</span><span class="p">&gt;();</span>
        <span class="p">}</span>
        <span class="c1">//Adds the graphic to the composition</span>
        <span class="k">public</span> <span class="k">void</span> <span class="nf">Add</span><span class="p">(</span><span class="n">IGraphic</span> <span class="n">graphic</span><span class="p">)</span>
        <span class="p">{</span>
            <span class="n">graphics</span><span class="p">.</span><span class="n">Add</span><span class="p">(</span><span class="n">graphic</span><span class="p">);</span>
        <span class="p">}</span>
        <span class="c1">//Adds multiple graphics to the composition</span>
        <span class="k">public</span> <span class="k">void</span> <span class="nf">AddRange</span><span class="p">(</span><span class="k">params</span> <span class="n">IGraphic</span><span class="p">[]</span> <span class="n">graphic</span><span class="p">)</span>
        <span class="p">{</span>
            <span class="n">graphics</span><span class="p">.</span><span class="n">AddRange</span><span class="p">(</span><span class="n">graphic</span><span class="p">);</span>
        <span class="p">}</span>
        <span class="c1">//Removes the graphic from the composition</span>
        <span class="k">public</span> <span class="k">void</span> <span class="nf">Delete</span><span class="p">(</span><span class="n">IGraphic</span> <span class="n">graphic</span><span class="p">)</span>
        <span class="p">{</span>
            <span class="n">graphics</span><span class="p">.</span><span class="n">Remove</span><span class="p">(</span><span class="n">graphic</span><span class="p">);</span>
        <span class="p">}</span>
        <span class="c1">//Prints the graphic.</span>
        <span class="k">public</span> <span class="k">void</span> <span class="nf">Print</span><span class="p">()</span>
        <span class="p">{</span>
            <span class="k">foreach</span> <span class="p">(</span><span class="kt">var</span> <span class="n">childGraphic</span> <span class="k">in</span> <span class="n">graphics</span><span class="p">)</span>
            <span class="p">{</span>
                <span class="n">childGraphic</span><span class="p">.</span><span class="n">Print</span><span class="p">();</span>
            <span class="p">}</span>
        <span class="p">}</span>
    <span class="p">}</span>
<span class="p">}</span>
<span class="c1">//////Ad</span>
</pre></div>
