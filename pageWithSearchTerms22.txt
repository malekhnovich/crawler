<p><i><b>Enterprise Integration Patterns</b></i> is a book by <a class="new" href="/w/index.php?title=Gregor_Hohpe&amp;action=edit&amp;redlink=1" title="Gregor Hohpe (page does not exist)">Gregor Hohpe</a> and <a class="new" href="/w/index.php?title=Bobby_Woolf&amp;action=edit&amp;redlink=1" title="Bobby Woolf (page does not exist)">Bobby Woolf</a> and describes 65 <a class="mw-redirect" href="/wiki/Design_pattern_(computer_science)" title="Design pattern (computer science)">design patterns</a> for the use of <a href="/wiki/Enterprise_application_integration" title="Enterprise application integration">enterprise application integration</a> and <a href="/wiki/Message-oriented_middleware" title="Message-oriented middleware">message-oriented middleware</a> in the form of a <a href="/wiki/Pattern_language" title="Pattern language">pattern language</a>.</p>

<h2><span class="mw-headline" id="Implementation">Implementation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Enterprise_Integration_Patterns&amp;action=edit&amp;section=1" title="Edit section: Implementation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><i>Enterprise Integration Patterns</i> are implemented in many open source <a href="/wiki/Enterprise_service_bus" title="Enterprise service bus">ESBs</a>, including <a href="/wiki/Spring_Integration" title="Spring Integration">Spring Integration</a>, WSO2 ESB, <a href="/wiki/Apache_Camel" title="Apache Camel">Apache Camel</a>, <a class="external text" href="https://www.redhat.com/en/technologies/jboss-middleware/fuse" rel="nofollow">Red Hat JBoss Fuse</a>, <a class="mw-redirect" href="/wiki/Mule_ESB" title="Mule ESB">Mule ESB</a> or <a href="/wiki/Guaran%C3%A1_DSL" title="Guaraná DSL">Guaraná DSL</a>. The book includes an icon-based pattern language, sometimes nicknamed "GregorGrams" after one of the authors.</p>
<h2><span class="mw-headline" id="Enterprise_Integration_Patterns">Enterprise Integration Patterns</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Enterprise_Integration_Patterns&amp;action=edit&amp;section=2" title="Edit section: Enterprise Integration Patterns">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The pattern language presented in the book consists of 65 patterns structured into 9 categories, which largely follow the flow of a message from one system to the next through channels, routing, and transformations.</p>
