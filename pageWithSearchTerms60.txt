<h1 class="firstHeading" id="firstHeading" lang="en">Interpreter pattern</h1>

<p>In <a href="/wiki/Computer_programming" title="Computer programming">computer programming</a>, the <b>interpreter pattern</b> is a <a class="mw-redirect" href="/wiki/Design_pattern_(computer_science)" title="Design pattern (computer science)">design pattern</a> that specifies how to evaluate sentences in a language. The basic idea is to have a <a class="mw-redirect" href="/wiki/Class_(computer_science)" title="Class (computer science)">class</a> for each symbol (<a class="mw-redirect" href="/wiki/Terminal_symbol" title="Terminal symbol">terminal</a> or <a class="mw-redirect" href="/wiki/Nonterminal_symbol" title="Nonterminal symbol">nonterminal</a>) in a <a class="mw-redirect" href="/wiki/Domain_specific_languages" title="Domain specific languages">specialized computer language</a>. The <a href="/wiki/Abstract_syntax_tree" title="Abstract syntax tree">syntax tree</a> of a sentence in the language is an instance of the <a href="/wiki/Composite_pattern" title="Composite pattern">composite</a> pattern and is used to evaluate (interpret) the sentence for a client.<sup class="reference" id="cite_ref-GoF_1-0"><a href="#cite_note-GoF-1">[1]</a></sup><sup class="reference" style="white-space:nowrap;">:243</sup> See also <a href="/wiki/Composite_pattern" title="Composite pattern">Composite pattern</a>.</p>

<h2><span class="mw-headline" id="Uses">Uses</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpreter_pattern&amp;action=edit&amp;section=1" title="Edit section: Uses">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul>
<li>Specialized database query languages such as <a href="/wiki/SQL" title="SQL">SQL</a>.</li>
<li>Specialized computer languages which are often used to describe communication protocols.</li>
<li>Most general-purpose computer languages actually incorporate several specialized languages.</li>
</ul>

<h2><span class="mw-headline" id="Example">Example</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpreter_pattern&amp;action=edit&amp;section=3" title="Edit section: Example">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span class="mw-headline" id="BNF">BNF</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpreter_pattern&amp;action=edit&amp;section=4" title="Edit section: BNF">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The following <a class="mw-redirect" href="/wiki/Backus-Naur_Form" title="Backus-Naur Form">Backus-Naur Form</a> example illustrates the interpreter pattern. The grammar</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
expression <span class="o">::=</span> plus | minus | variable | number
plus <span class="o">::=</span> expression expression '+'
minus <span class="o">::=</span> expression expression '-'
variable  <span class="o">::=</span> 'a' | 'b' | 'c' | ... | 'z'
digit = '0' | '1' | ... | '9'
number <span class="o">::=</span> digit | digit number
</pre></div>
<p>defines a language which contains <a class="mw-redirect" href="/wiki/Reverse_Polish_Notation" title="Reverse Polish Notation">Reverse Polish Notation</a> expressions like:</p>
<pre>
a b +
a b c + -
a b + c a - -
</pre>
<h3><span class="mw-headline" id="C.23">C#</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Interpreter_pattern&amp;action=edit&amp;section=5" title="Edit section: C#">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>This structural code demonstrates the Interpreter patterns, which using a defined grammar, provides the interpreter that processes parsed statements.</p>

    <span class="o">}</span>
<span class="o">}</span>

<span class="kd">class</span> <span class="nc">Variable</span> <span class="kd">implements</span> <span class="n">Expression</span> <span class="o">{</span>
    <span class="kd">private</span> <span class="n">String</span> <span class="n">name</span><span class="o">;</span>
    <span class="kd">public</span> <span class="nf">Variable</span><span class="o">(</span><span class="n">String</span> <span class="n">name</span><span class="o">)</span>       <span class="o">{</span> <span class="k">this</span><span class="o">.</span><span class="na">name</span> <span class="o">=</span> <span class="n">name</span><span class="o">;</span> <span class="o">}</span>
    <span class="kd">public</span> <span class="kt">int</span> <span class="nf">interpret</span><span class="o">(</span><span class="n">Map</span><span class="o">&lt;</span><span class="n">String</span><span class="o">,</span><span class="n">Expression</span><span class="o">&gt;</span> <span class="n">variables</span><span class="o">)</span>  <span class="o">{</span> 
        <span class="k">if</span><span class="o">(</span><span class="kc">null</span><span class="o">==</span><span class="n">variables</span><span class="o">.</span><span class="na">get</span><span class="o">(</span><span class="n">name</span><span class="o">))</span> <span class="k">return</span> <span class="mi">0</span><span class="o">;</span> <span class="c1">//Either return new Number(0).</span>
        <span class="k">return</span> <span class="n">variables</span><span class="o">.</span><span class="na">get</span><span class="o">(</span><span class="n">name</span><span class="o">).</span><span class="na">interpret</span><span class="o">(</span><span class="n">variables</span><span class="o">);</span> 
    <span class="o">}</span>
<span class="o">}</span>
</pre></div>
<p>While the interpreter pattern does not address parsing<sup class="reference" id="cite_ref-GoF_1-1"><a href="#cite_note-GoF-1">[1]</a></sup><sup class="reference" style="white-space:nowrap;">:247</sup> a parser is provided for completeness.</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="kn">import</span> <span class="nn">java.util.Map</span><span class="o">;</span>
<span class="kn">import</span> <span class="nn">java.util.Stack</span><span class="o">;</span>

<span class="kd">class</span> <span class="nc">Evaluator</span> <span class="kd">implements</span> <span class="n">Expression</span> <span class="o">{</span>
    <span class="kd">private</span> <span class="n">Expression</span> <span class="n">syntaxTree</span><span class="o">;</span>
 
    <span class="kd">public</span> <span class="nf">Evaluator</span><span class="o">(</span><span class="n">String</span> <span class="n">expression</span><span class="o">)</span> <span class="o">{</span>
        <span class="n">Stack</span><span class="o">&lt;</span><span class="n">Expression</span><span class="o">&gt;</span> <span class="n">expressionStack</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Stack</span><span class="o">&lt;</span><span class="n">Expression</span><span class="o">&gt;();</span>
        <span class="k">for</span> <span class="o">(</span><span class="n">String</span> <span class="n">token</span> <span class="o">:</span> <span class="n">expression</span><span class="o">.</span><span class="na">split</span><span class="o">(</span><span class="s">" "</span><span class="o">))</span> <span class="o">{</span>
            <span class="k">if</span>  <span class="o">(</span><span class="n">token</span><span class="o">.</span><span class="na">equals</span><span class="o">(</span><span class="s">"+"</span><span class="o">))</span> <span class="o">{</span>
                <span class="n">Expression</span> <span class="n">subExpression</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Plus</span><span class="o">(</span><span class="n">expressionStack</span><span class="o">.</span><span class="na">pop</span><span class="o">(),</span> <span class="n">expressionStack</span><span class="o">.</span><span class="na">pop</span><span class="o">());</span>
                <span class="n">expressionStack</span><span class="o">.</span><span class="na">push</span><span class="o">(</span> <span class="n">subExpression</span> <span class="o">);</span>
            <span class="o">}</span>
            <span class="k">else</span> <span class="k">if</span> <span class="o">(</span><span class="n">token</span><span class="o">.</span><span class="na">equals</span><span class="o">(</span><span class="s">"-"</span><span class="o">))</span> <span class="o">{</span>
                <span class="c1">// it's necessary remove first the right operand from the stack</span>
                <span class="n">Expression</span> <span class="n">right</span> <span class="o">=</span> <span class="n">expressionStack</span><span class="o">.</span><span class="na">pop</span><span class="o">();</span>
                <span class="c1">// ..and after the left one</span>
                <span class="n">Expression</span> <span class="n">left</span> <span class="o">=</span> <span class="n">expressionStack</span><span class="o">.</span><span class="na">pop</span><span class="o">();</span>
                <span class="n">Expression</span> <span class="n">subExpression</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Minus</span><span class="o">(</span><span class="n">left</span><span class="o">,</span> <span class="n">right</span><span class="o">);</span>
                <span class="n">expressionStack</span><span class="o">.</span><span class="na">push</span><span class="o">(</span> <span class="n">subExpression</span> <span class="o">);</span>
            <span class="o">}</span>
            <span class="k">else</span>                        
                <span class="n">expressionStack</span><span class="o">.</span><span class="na">push</span><span class="o">(</span> <span class="k">new</span> <span class="n">Variable</span><span class="o">(</span><span class="n">token</span><span class="o">)</span> <span class="o">);</span>
        <span class="o">}</span>
        <span class="n">syntaxTree</span> <span class="o">=</span> <span class="n">expressionStack</span><span class="o">.</span><span class="na">pop</span><span class="o">();</span>
    <span class="o">}</span>
 
    <span class="kd">public</span> <span class="kt">int</span> <span class="nf">interpret</span><span class="o">(</span><span class="n">Map</span><span class="o">&lt;</span><span class="n">String</span><span class="o">,</span><span class="n">Expression</span><span class="o">&gt;</span> <span class="n">context</span><span class="o">)</span> <span class="o">{</span>
        <span class="k">return</span> <span class="n">syntaxTree</span><span class="o">.</span><span class="na">interpret</span><span class="o">(</span><span class="n">context</span><span class="o">);</span>
    <span class="o">}</span>
<span class="o">}</span>
</pre></div>
<p>Finally evaluating the expression "w x z - +" with w = 5, x = 10, and z = 42.</p>

<pre>
<span class="kn">import</span> <span class="nn">java.util.Map</span><span class="o">;</span>
<span class="kn">import</span> <span class="nn">java.util.HashMap</span><span class="o">;</span>

<span class="kd">public</span> <span class="kd">class</span> <span class="nc">InterpreterExample</span> <span class="o">{</span>
    <span class="kd">public</span> <span class="kd">static</span> <span class="kt">void</span> <span class="nf">main</span><span class="o">(</span><span class="n">String</span><span class="o">[]</span> <span class="n">args</span><span class="o">)</span> <span class="o">{</span>
        <span class="n">String</span> <span class="n">expression</span> <span class="o">=</span> <span class="s">"w x z - +"</span><span class="o">;</span>
        <span class="n">Evaluator</span> <span class="n">sentence</span> <span class="o">=</span> <span class="k">new</span> <span class="n">Evaluator</span><span class="o">(</span><span class="n">expression</span><span class="o">);</span>
        <span class="n">Map</span><span class="o">&lt;</span><span class="n">String</span><span class="o">,</span><span class="n">Expression</span><span class="o">&gt;</span> <span class="n">variables</span> <span class="o">=</span> <span class="k">new</span> <span class="n">HashMap</span><span class="o">&lt;</span><span class="n">String</span><span class="o">,</span><span class="n">Expression</span><span class="o">&gt;();</span>
        <span class="n">variables</span><span class="o">.</span><span class="na">put</span><span class="o">(</span><span class="s">"w"</span><span class="o">,</span> <span class="k">new</span> <span class="n">Number</span><span class="o">(</span><span class="mi">5</span><span class="o">));</span>
        <span class="n">variables</span><span class="o">.</span><span class="na">put</span><span class="o">(</span><span class="s">"x"</span><span class="o">,</span> <span class="k">new</span> <span class="n">Number</span><span class="o">(</span><span class="mi">10</span><span class="o">));</span>
        <span class="n">variables</span><span class="o">.</span><span class="na">put</span><span class="o">(</span><span class="s">"z"</span><span class="o">,</span> <span class="k">new</span> <span class="n">Number</span><span class="o">(</span><span class="mi">42</span><span class="o">));</span>
        <span class="kt">int</span> <span class="n">result</span> <span class="o">=</span> <span class="n">sentence</span><span class="o">.</span><span class="na">interpret</span><span class="o">(</span><span class="n">variables</span><span class="o">);</span>
        <span class="n">System</span><span class="o">.</span><span class="na">out</span><span class="o">.</span><span class="na">println</span><span class="o">(</span><span class="n">result</span><span class="o">);</span>
    <span class="o">}</span>
<span class="o">}</span>
