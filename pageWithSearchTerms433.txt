<h1 class="firstHeading" id="firstHeading" lang="en">Graphical user interface builder</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><p>A <b>graphical user interface builder</b> (or <b>GUI builder</b>), also known as <b>GUI designer</b>, is a <a class="mw-redirect" href="/wiki/Software_development_tool" title="Software development tool">software development tool</a> that simplifies the creation of <a href="/wiki/Graphical_user_interface" title="Graphical user interface">GUIs</a> by allowing the designer to arrange <a class="mw-redirect" href="/wiki/Graphical_control_element" title="Graphical control element">graphical control elements</a> (often called widgets) using a <a class="mw-redirect" href="/wiki/Drag-and-drop" title="Drag-and-drop">drag-and-drop</a> <a href="/wiki/WYSIWYG" title="WYSIWYG">WYSIWYG</a> editor. Without a GUI builder, a GUI must be built by manually specifying each widget's parameters in source-code, with no visual feedback until the program is run.</p>
<p>User interfaces are commonly programmed using an <a href="/wiki/Event-driven_programming" title="Event-driven programming">event-driven architecture</a>, so GUI builders also simplify creating event-driven code. This supporting code connects widgets with the outgoing and incoming <a href="/wiki/Event_(computing)" title="Event (computing)">events</a> that trigger the functions providing the application logic.</p>
<p>Some graphical user interface builders, such as e.g. <a href="/wiki/Glade_Interface_Designer" title="Glade Interface Designer">Glade Interface Designer</a>, automatically generate all the <a href="/wiki/Source_code" title="Source code">source code</a> for a <a class="mw-redirect" href="/wiki/Graphical_control_element" title="Graphical control element">graphical control element</a>. Others, like <a href="/wiki/Interface_Builder" title="Interface Builder">Interface Builder</a>, generate <a href="/wiki/Serialization" title="Serialization">serialized</a> object instances that are then loaded by the application.</p>
