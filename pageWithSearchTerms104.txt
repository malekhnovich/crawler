<h1 class="firstHeading" id="firstHeading" lang="en">Helper class</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><p>In <a href="/wiki/Object-oriented_programming" title="Object-oriented programming">object-oriented programming</a>, a <b>helper class</b> is used to assist in providing some functionality, which isn't the main goal of the application or class in which it is used<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="There's no sources backing this up. (November 2015)">citation needed</span></a></i>]</sup>. An instance of a helper class is called a <b>helper object</b> (for example, in the <a href="/wiki/Delegation_pattern" title="Delegation pattern">delegation pattern</a>).</p>
<p>Helper classes are often created in introductory programming lessons, after the novice programmer has moved beyond creating one or two classes.</p>
<p>A <a href="/wiki/Utility_class" title="Utility class">utility class</a> is a special case of a helper class in which the methods are all static. In general, helper classes do not have to have all static methods, and may have instance variables and multiple instances of the helper class may exist.</p>
<h2><span class="mw-headline" id="Example">Example</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Helper_class&amp;action=edit&amp;section=1" title="Edit section: Example">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>This is also an example of a <a href="/wiki/Utility_class" title="Utility class">utility class</a>.</p>

    <span class="p">{</span>
        <span class="k">return</span> <span class="s">"Meow meow "</span> <span class="p">+</span> <span class="n">text</span> <span class="p">+</span> <span class="s">"!"</span><span class="p">;</span>
    <span class="p">}</span>

    <span class="k">public</span> <span class="k">static</span> <span class="kt">string</span> <span class="nf">WoofPrepend</span><span class="p">(</span><span class="kt">string</span> <span class="n">text</span><span class="p">)</span>
    <span class="p">{</span>
        <span class="k">return</span> <span class="s">"Woof woof "</span> <span class="p">+</span> <span class="n">text</span> <span class="p">+</span> <span class="s">"!"</span><span class="p">;</span>
    <span class="p">}</span>

    <span class="k">public</span> <span class="k">static</span> <span class="kt">string</span> <span class="nf">WoohPrepend</span><span class="p">(</span><span class="kt">string</span> <span class="n">text</span><span class="p">)</span>
