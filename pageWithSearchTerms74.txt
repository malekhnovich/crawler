<h1 class="firstHeading" id="firstHeading" lang="en">Concurrency pattern</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">

</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><p>In <a href="/wiki/Software_engineering" title="Software engineering">software engineering</a>, <b>concurrency patterns</b> are those types of <a class="mw-redirect" href="/wiki/Design_pattern_(computer_science)" title="Design pattern (computer science)">design patterns</a> that deal with the <a class="mw-redirect" href="/wiki/Multi-threaded" title="Multi-threaded">multi-threaded</a> programming paradigm. Examples of this class of patterns include:</p>
<ul>
<li><a class="mw-redirect" href="/wiki/Active_Object" title="Active Object">Active Object</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup></li>
<li><a href="/wiki/Balking_pattern" title="Balking pattern">Balking pattern</a></li>
<li><a href="/wiki/Barrier_(computer_science)" title="Barrier (computer science)">Barrier</a></li>
<li><a href="/wiki/Double-checked_locking" title="Double-checked locking">Double-checked locking</a></li>
<li><a href="/wiki/Guarded_suspension" title="Guarded suspension">Guarded suspension</a></li>
<li>Leaders/followers pattern</li>
<li><a href="/wiki/Monitor_(synchronization)" title="Monitor (synchronization)">Monitor Object</a></li>
<li><a href="/wiki/Reactor_pattern" title="Reactor pattern">Reactor pattern</a></li>
<li><a class="mw-redirect" href="/wiki/Read_write_lock_pattern" title="Read write lock pattern">Read write lock pattern</a></li>
<li><a class="mw-redirect" href="/wiki/Scheduler_pattern" title="Scheduler pattern">Scheduler pattern</a></li>
<li><a class="mw-redirect" href="/wiki/Thread_pool_pattern" title="Thread pool pattern">Thread pool pattern</a></li>
<li><a href="/wiki/Thread-local_storage" title="Thread-local storage">Thread-local storage</a></li>
</ul>
