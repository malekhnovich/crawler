<p>In <a href="/wiki/Computer_programming" title="Computer programming">computer programming</a>, a <b>Type Tunnel <a class="mw-redirect" href="/wiki/Design_pattern_(computer_science)" title="Design pattern (computer science)">pattern</a></b> is where a group of physically unrelated types may be tunneled through an extensible adaptation layer and presented in unified form to an underlying layer for manipulation as a whole. It consists of the following:</p>
<ol>
<li>a generic, extensible <a class="mw-redirect" href="/wiki/Interface_(computer_science)" title="Interface (computer science)">interface</a> layer, used in client code, which can interact with heterogeneous types, and</li>
<li>a tunnel mechanism, which translates between the heterogeneous types expressed in the client code into the <a href="/wiki/Type_system" title="Type system">type</a> understood by</li>
<li>a concrete <a class="mw-redirect" href="/wiki/API" title="API">API</a> layer, which manipulates a single concrete type.</li>
</ol>
<p>Tunnel mechanism include <a href="/wiki/Shim_(computing)" title="Shim (computing)">Shims</a> and conversion <a href="/wiki/Constructor_(object-oriented_programming)" title="Constructor (object-oriented programming)">constructors</a>.</p>

<h3><span class="mw-headline" id="C.2B.2B">C++</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Type_Tunnel_pattern&amp;action=edit&amp;section=2" title="Edit section: C++">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Example that uses Shims as tunnel mechanism.</p>
<div class="mw-highlight mw-content-ltr" dir="ltr">
<pre>
<span class="c1">// 1. Interface layer</span>
<span class="k">template</span> <span class="o">&lt;</span><span class="k">typename</span> <span class="n">S</span><span class="o">&gt;</span> 
<span class="kt">void</span> <span class="n">foo</span><span class="p">(</span><span class="n">S</span> <span class="n">s</span><span class="p">)</span> 
<span class="p">{</span>    
    <span class="n">bar</span><span class="p">(</span><span class="n">to_cstr_ptr</span><span class="p">(</span><span class="n">s</span><span class="p">));</span> 
<span class="p">}</span>

<span class="c1">// 2. Tunnel mechanism: Shim</span>
<span class="kt">char</span> <span class="k">const</span><span class="o">*</span> <span class="n">to_cstr_ptr</span><span class="p">(</span><span class="kt">int</span><span class="p">)</span> <span class="p">{</span> <span class="p">...</span> <span class="p">}</span>
<span class="kt">char</span> <span class="k">const</span><span class="o">*</span> <span class="n">to_cstr_ptr</span><span class="p">(</span><span class="kt">char</span> <span class="k">const</span><span class="o">*</span><span class="p">)</span> <span class="p">{</span> <span class="p">...</span> <span class="p">}</span>
<span class="kt">char</span> <span class="k">const</span><span class="o">*</span> <span class="n">to_cstr_ptr</span><span class="p">(</span><span class="n">std</span><span class="o">::</span><span class="n">string</span><span class="p">)</span> <span class="p">{</span> <span class="p">...</span> <span class="p">}</span>

<span class="c1">// 3. Concrete API layer</span>
<span class="kt">void</span> <span class="n">bar</span><span class="p">(</span><span class="kt">char</span> <span class="k">const</span><span class="o">*</span><span class="p">)</span> <span class="p">{</span> <span class="p">...</span> <span class="p">}</span>

<span class="c1">// Usage</span>
<span class="kt">int</span> <span class="n">main</span><span class="p">()</span> 
<span class="p">{</span>
    <span class="n">foo</span><span class="p">(</span><span class="mi">123</span><span class="p">);</span> 
    <span class="n">foo</span><span class="p">(</span><span class="s">"a C string"</span><span class="p">);</span> 
    <span class="n">foo</span><span class="p">(</span><span class="n">std</span><span class="o">::</span><span class="n">string</span><span class="p">(</span><span class="s">"a std::string"</span><span class="p">));</span> 
<span class="p">}</span>
