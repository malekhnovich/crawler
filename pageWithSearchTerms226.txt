<h1 class="firstHeading" id="firstHeading" lang="en">Follow-the-sun</h1>
<div class="mw-body-content" id="bodyContent">
<div id="siteSub">From Wikipedia, the free encyclopedia</div>
<div id="contentSub"></div>
<div class="mw-jump" id="jump-to-nav">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
</div>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><p><b>Follow-the-sun</b> is a type of global <a href="/wiki/Workflow" title="Workflow">workflow</a> in which tasks are passed around daily between work sites that are many time zones apart. Such a workflow is set up in order to reduce project duration and increase responsiveness. Thus, the work is "following the sun" and never stops.</p>
<p>For example, at the end of the day, a systems support team in <a href="/wiki/Silicon_Valley" title="Silicon Valley">Silicon Valley</a> will pass its work tasks to a support team in <a class="mw-redirect" href="/wiki/Bangalore,_India" title="Bangalore, India">Bangalore, India</a>, which, at the end of its day, passes its work to <a href="/wiki/Paris" title="Paris">Paris</a>, which passes it back to <a href="/wiki/Silicon_Valley" title="Silicon Valley">Silicon Valley</a>.</p>
<p>This workflow is more complex to implement than local software support due to teams collaborating at a distance. It is also discussed in software development, but is more difficult to achieve because of coordination costs.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup></p>
<p></p>

<h2><span class="mw-headline" id="Follow-the-moon">Follow-the-moon</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Follow-the-sun&amp;action=edit&amp;section=1" title="Edit section: Follow-the-moon">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>A related concept is <i>follow-the-moon</i>, which is scheduling work to be performed specifically during local night-time hours in order to, for instance, save on <a class="mw-redirect" href="/wiki/Datacenter" title="Datacenter">datacenter</a> costs by using <a href="/wiki/Peak_demand" title="Peak demand">cheaper night-time electricity</a><sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> or spare processing power.</p>
